## Использование

Чтобы запустить программу, необходимо перейти в папку с ней и в консоль ввести команду:

```console
lua54.exe fano.lua
```

Появляется ввод режима вводы данных

```console
1. Ввод строки
2. Посимвольный ввод
```

Режим вводе целой строки для кодирования

